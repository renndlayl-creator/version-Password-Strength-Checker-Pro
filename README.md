# Password Strength Checker Pro

Aplicación desarrollada en Python con Tkinter para analizar la seguridad de contraseñas y generar contraseñas seguras.

## Características

* Análisis de fortaleza de contraseñas.
* Generador de contraseñas seguras aleatorias.
* Cálculo de entropía (bits de seguridad).
* Historial de análisis con fecha y hora.
* Copiar contraseña al portapapeles.
* Mostrar/Ocultar contraseña.
* Exportación de historial a CSV.
* Interfaz con tema oscuro rosa.

## Tecnologías

* Python 3
* Tkinter
* CSV
* Regex (re)

## Instalación

```bash
git clone https://github.com/renndlayl-creator/version-Password-Strength-Checker-Pro.git
cd version-Password-Strength-Checker-Pro
python password_checker_pro.py
```
